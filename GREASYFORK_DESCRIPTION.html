B站 / YouTube 首页 AI 视频过滤器

厌倦了 B 站或 YouTube 首页反复出现的擦边、猎奇、标题党、焦虑营销或其他不想看的推荐？这个脚本允许你直接用一句自然语言描述偏好，再结合本地规则、手动“不喜欢”样本和 AI 语义判断，逐步整理出更清爽的首页。

脚本不会因为 AI 一次判断就永久拉黑整个创作者。AI 命中时只隐藏当前视频；同一创作者多次命中后，只会显示拉黑建议，最终决定权仍在用户手中。

主要功能

  • 使用自然语言描述不想看到的内容。
  • 支持 B 站与 YouTube 首页首屏、无限滚动，以及动态出现的新视频。
  • 本地标题关键词、正则规则、创作者黑名单和白名单优先判断。
  • 只有本地规则和缓存无法确定时才调用 AI，减少 API token 消耗。
  • 在视频卡片上点击右键，可标记“不喜欢此视频”、手动拉黑创作者或将创作者加入白名单。
  • 手动标记后，AI 可以分析标题特征并更新本地偏好画像。
  • AI 可提炼候选标题规则，但必须由用户确认后才会启用。
  • 浮动暂停按钮可停止首页自动 AI 监视，本地规则和已有缓存仍继续生效。
  • 只观察首页推荐区域，离开首页后停止卡片扫描和自动 AI 队列。
  • 危险正则会被拒绝保存，避免复杂回溯造成网页卡顿。
  • 支持配置与黑白名单导入、导出；导出文件不包含 API Key 和 AI 判断缓存。
  • API 失败时保持视频显示，不会因为网络或模型故障误隐藏内容。

支持的 AI 服务商

当前支持 DeepSeek、AiHubMix、OpenAI、Google Gemini、Anthropic Claude、豆包（火山方舟）、阿里云百炼（通义千问）、智谱 GLM、Kimi（月之暗面）、腾讯混元和百度千帆（ERNIE）。

脚本会自动处理各平台的请求地址、鉴权头、输出 token 参数和响应格式。每个平台的模型名与 API Key 分开保存在当前 Tampermonkey 脚本的本地存储中，切换服务商不会覆盖其他平台配置。

如何使用

  1. 安装 Tampermonkey 和本脚本，打开或刷新 B 站或 YouTube 首页。
  2. 点击右下角“AI 过滤”按钮。
  3. 填写要过滤的内容描述。
  4. 展开“API 与备份”，选择服务商，填写该平台的 API Key，并点击“测试连接”。
  5. 如需语义过滤，勾选「启用自动 AI 监视」并确认风险提示（默认关闭）。
  6. 保存设置后，本地规则会立即生效；仅在开启自动监视后才会把模糊内容发给 AI。
  7. 遇到特别不喜欢的视频时，在卡片上点击右键并选择“不喜欢此视频 · 隐藏并让 AI 学习”。

如何节省 token

  • 白名单、黑名单、标题关键词和正则全部在浏览器本地运行。
  • 匹配与不匹配结果都会缓存，视频重新出现时不重复询问 AI。
  • 一次最多批量判断 10 个视频，减少重复提示词。
  • 限制标题、创作者名称、偏好画像和历史样本的发送长度。
  • 输出额度按视频数量动态计算，只有 JSON 为空或被截断时才提高额度补救一次。
  • 对支持的平台默认关闭不必要的深度思考或联网增强。
  • 自动 AI 监视默认关闭；可在设置中开启，开启前会提示隐私风险。
  • 关闭自动监视时，手动右键添加的不喜欢样本仍可按用户明确操作进行学习。

隐私与安全

自动 AI 监视默认关闭。只有你开启自动监视且本地规则/缓存无法判断，或主动学习/测试连接时，脚本才会把过滤描述、偏好画像、近期不喜欢样本标题、待判断视频标题和页面显示的创作者名称发送给你选择的 API 服务商。脚本不会主动发送 B 站或 YouTube Cookie、账号资料、频道标识或页面正文。

API Key 仅保存在当前 Tampermonkey 脚本的本地存储中，不会写入导出文件。请注意，浏览器本地存储不等同于系统级密码保险箱，请使用单独创建、额度受控的 API Key。

当前范围

当前版本处理 https://www.bilibili.com/ 和 https://www.youtube.com/ 首页。YouTube 仅处理标准视频推荐；Shorts 与广告会跳过。频道标识缺失的卡片不进入自动 AI 队列，但在视频 ID 与标题可用时可右键手动隐藏。搜索页、订阅页、排行榜、视频播放页、B站 App、观看历史和自动云同步均不处理。按住 Shift 再点击右键，可以使用浏览器原生右键菜单。
