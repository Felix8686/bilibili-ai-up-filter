B站首页 AI UP 主过滤器

当前版本：v0.2.0

这是一个 Tampermonkey 用户脚本。它会读取 B 站首页新出现的推荐卡片，批量调用用户配置的 AI 接口进行语义判断；命中后将对应 UP 主的 UID 保存到本地黑名单，并隐藏该 UP 主当前及之后出现的首页推荐。也可以在视频卡片上点击右键，手动标记不喜欢的视频，让 AI 从这些样本中持续学习过滤偏好。

当前版本范围

  • 只处理 https://www.bilibili.com/ 首页。
  • 支持首页首屏、无限滚动和“换一换”产生的新卡片。
  • 支持在首页视频卡片上右键添加不喜欢样本，并自动生成 AI 偏好画像。
  • 不处理搜索页、排行榜、视频页、B站 App 或观看历史。
  • 判断失败时保持视频显示，不会因为 API 故障自动拉黑。

安装

  1. 在 Chrome 或 Edge 安装 Tampermonkey。
  2. 在 Tampermonkey 中新建脚本。
  3. 复制 bilibili-ai-up-filter.user.js 的全部内容并保存。
  4. 打开或刷新 B 站首页，右下角会显示“AI 过滤”按钮。

配置

  1. 点击页面右下角“AI 过滤”，或使用 Tampermonkey 菜单中的“打开 B站首页 AI 过滤设置”。
  2. 填写要过滤的内容描述，例如“擦边、夸张猎奇、标题党的内容”。
  3. 选择 DeepSeek 或 AiHubMix，填写模型名和对应的 API Key。
  4. 点击“测试连接”，确认接口可以返回脚本要求的 JSON。
  5. 点击“保存”，脚本会开始判断首页推荐。

DeepSeek 默认模型为 deepseek-v4-flash，AiHubMix 默认模型为 gpt-4o-mini。如果服务商调整模型名称，可以直接在设置中修改。

右键标记不喜欢与主动学习

  1. 在 B 站首页的视频卡片上点击右键。
  2. 选择“不喜欢此视频 · 隐藏并让 AI 学习”。
  3. 该 BVID 会立即进入本地不喜欢样本并从首页隐藏。
  4. 配置了 API Key 时，AI 会自动分析标题和 UP 主名称，提炼内容特征并更新偏好画像。
  5. 后续首页推荐会同时参考手写过滤描述、偏好画像和近期不喜欢样本。

手动标记只隐藏当前视频，不会直接拉黑整个 UP 主。按住 Shift 再点击右键，可以使用浏览器原生右键菜单。未配置 API Key 时样本仍会保存，保存有效配置后会自动补充分析。

黑名单与备份

  • 黑名单使用 UP 主 UID 作为唯一键，保存在 Tampermonkey 本地存储中。
  • 手动不喜欢样本使用 BVID 作为唯一键，与 UP 主黑名单分开保存。
  • 可在设置面板中删除误判的 UP 主。
  • “导出”会保存过滤描述、非敏感设置、UP 主黑名单、不喜欢样本和学习画像。
  • 导出文件不会包含 API Key。
  • “导入”会按 UID 和 BVID 合并数据，并保留当前设备的 API Key；v0.1.0 导出的旧备份仍可导入。

隐私说明

AI 判断和主动学习会把过滤描述、偏好画像、不喜欢样本的视频标题及页面显示的 UP 主名称发送给所选 API 服务商。脚本不会主动发送 B 站 Cookie、账号资料或页面正文。API Key 仅存放在当前 Tampermonkey 脚本存储中，但浏览器本地存储不等同于系统级密码保险箱。
